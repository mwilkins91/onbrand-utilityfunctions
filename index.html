doIfTag

Loop over each tile, and if the tag specified (1st param) is present, execute yesTagFn (2nd param) if the tag is not present, execute noTagFn (3rd param). In both callbacks, this reffers to the tile currently being checked

doIfTag(filterBy: string, yesTagFn: function, noTagFn: function, not: any, OPTIONAL: string): boolean
Parameters
filterBy (string) --> The tag to look for
yesTagFn (function = function(){}) --> The function to run if a tile has the desired tag.
noTagFn (function = function(){}) --> The function to run if a tile does NOT have the desired tag.
not (any = '')
OPTIONAL (string) : A css class to ignore, and to add after the tile has been processed.
Returns
boolean: --> returns true if the front end tags are enabled, otherwise returns false and logs an error.
